================================================================================
              Word 模板自动填写工具 - 快速开始
================================================================================

🚀 两步安装，立即使用！

【第一步】下载依赖
  双击运行 → 下载依赖.bat
  等待提示"依赖下载完成"

【第二步】打开工具
  双击打开 → index.html

【第三步】设置工作区（推荐）
  1. 点击左下角 📁 "设置工作区"
  2. 选择一个文件夹
  3. 授予权限
  4. 完成！


================================================================================
                        基本使用流程
================================================================================

1️⃣ 创建模板
   - 点击"新建模板"
   - 上传 Word 模板文件
   - 配置字段和明细表
   - 保存

2️⃣ 填写表单
   - 点击"开始填写"
   - 填写数据
   - 点击"生成 Word"

3️⃣ 管理历史
   - 查看历史记录
   - 导出数据
